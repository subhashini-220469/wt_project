# 🚀 SMARTHIRE - React Frontend

A premium React-based dashboard for the AI Resume Screening System. Built with **Vite**, **Framer Motion**, and **Lucide React**.

## ✨ Features
- **Modern Dashboard**: Clean sidebar navigation and responsive layout.
- **Dynamic Upload**: Drag & drop support for multiple resumes.
- **Real-time Feedback**: Glassmorphism scanning overlay with processing status.
- **AI Ranking**: Direct integration with the FastAPI backend to display match scores.

## 🛠️ Setup Instructions

1. **Prerequisites**: Ensure you have [Node.js](https://nodejs.org/) installed.
2. **Navigate to directory**:
   ```powershell
   cd frontend-react
   ```
3. **Install Dependencies**:
   ```powershell
   npm install
   ```
4. **Run Development Server**:
   ```powershell
   npm run dev
   ```
5. **Access the App**: Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🔗 Backend Connection
The app connects to the backend at `http://localhost:8000`. 
Ensure your **FastAPI** server is running before clicking "Analyze".

## 🎨 Technology Stack
- **Framework**: React 18
- **Styling**: Vanilla CSS (Premium Custom Theme)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite
